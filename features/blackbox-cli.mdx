# BLACKBOX CLI

> Access the full power of BLACKBOXAI directly through a CLI

## Installation

You can install it for any device with the following command:

For Bash:

`curl -fsSL https://shell.blackbox.ai/api/scripts/omni-cli/download.sh | bash`

or PowerShell:

`Invoke-WebRequest -Uri "https://shell.blackbox.ai/api/scripts/omni-cli/download.ps1" -OutFile "download.ps1"; .\download.ps1`

**"The best AI agent for builders"** - everything you need is just one click away from your browser.

## Steps to setup:

You can simply provide a BLAKCBOX API Key from [here](https://www.blackbox.ai/dashboard) and select any of the best closed source or open source models provided through the API to power the BLACKBOX Agent

1. Run `blackbox`
2. Provide BLACKBOX API Key
3. Select from the best closed source or open source models provided through the API
4. Send your first task
