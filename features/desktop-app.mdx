## Overview

The BLACKBOXAI On-machine Agent provides a focused, powerful, and integrated environment for all your AI-assisted coding needs. It bundles the BLACKBOXAI IDE and all our core AI features into a single, native application, ensuring optimal performance and a seamless user experience. If you want a dedicated space for your development work, the Desktop App is the perfect choice.

BLACKBOXAI Agent integrates with over 35 IDEs, including VSCode, PyCharm, IntelliJ, Android Studio, Xcode, and many more. You can give the agent fullstack, front end, database, ML, data science tasks. You can have the agent implement several tasks with full autonomy or with customized autonomy or with 100% manual approval. The agent can read, write, edit files, run terminal commands, read logs, handle desktop apps etc.

## Key Features

- **Optimized Performance:** As a native application, it's faster and more responsive than web-based alternatives.
- **Cross-IDE Integration:** 35 IDEs, including VSCode, PyCharm, IntelliJ, Android Studio, Xcode, and many more.
- **Cross-Platform:** Available for Windows, macOS, and major Linux distributions.
- **Offline Capabilities:** Access certain features and manage your projects even when you have an unstable internet connection.

## Getting Started

1. **Download:** Visit the official website [blackbox.ai](https://www.blackbox.ai/) and get the installer for your operating system.
2. **Install:** Run the installer and follow the on-screen instructions.
3. **Launch & Sign In:** Open the BLACKBOXAI app from your applications folder and log in with your account.
4. **Open a Project:** Create a new project or open an existing folder to begin coding with the full power of BLACKBOXAI at your fingertips.
