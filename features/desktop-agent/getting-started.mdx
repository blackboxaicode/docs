Follow these simple steps to get up and running with the BLACKBOXAI Desktop App.
### 1. Download

Visit the official website [blackbox.ai](https://www.blackbox.ai/) and get the installer for your operating system.

![Download Button](/images/desktop-app/desktop_app-download-button.png)

**Supported Platforms:**
- Windows 10/11 (64-bit)
- macOS 10.15+ (Intel and Apple Silicon)
- Linux distributions (Ubuntu, Fedora, Debian, etc.)

### 2. Install

Run the installer and follow the on-screen instructions:

**Windows:**
1. Double-click the `.exe` installer
2. Follow the installation wizard
3. Choose installation directory (optional)
4. Complete the installation

**macOS:**
1. Open the `.dmg` file
2. Drag BLACKBOXAI to the Applications folder
![Drag and Drop](/images/desktop-app/desktop_app-drag-drop-to-applications.png)
3. Launch from Applications

**Linux:**
1. Install using your package manager
2. Or run the AppImage directly
3. Or use the provided `.deb`/`.rpm` packages

### 3. Launch & Sign In

1. Open the BLACKBOXAI app from your applications folder
2. Log in with your existing BLACKBOXAI account
3. If you don't have an account, create one for free

![Desktop App Home Screen](/images/desktop-app/desktop_app-Home.jpeg)

### 4. Start Working

Automate your daily tasks and workflows with the full power of BLACKBOXAI. Whether you're managing files, organizing documents, or working on projects, BlackBox is ready to help.

**Ways to start:**
- **Open Folder:** Connect to any folder to manage files and documents
- **New Project:** Create fresh projects for any type of work
- **Add Extensions:** Enhance capabilities with additional tools and integrations
- **Start Chatting:** Simply describe what you want to accomplish in plain language.
